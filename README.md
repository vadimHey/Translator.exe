# Транслятор с C-подобного языка на Python

## Описание проекта

Проект представляет собой транслятор, выполняющий перевод программ с ограниченного C-подобного языка в эквивалентный код на Python.
Транслятор реализован с использованием лексического анализатора (Flex) и синтаксического анализатора (Bison).

---

## Поддерживаемые конструкции входного языка

Входной язык ограничен синтаксисом C-подобного языка. Поддерживаются только следующие конструкции:

- объявление функции без параметров: `void main() { ... }`
- типы данных: `int`, `double`, `float`, `string`, `bool`
- объявление и инициализация переменных
- арифметические выражения
- логические выражения
- операторы присваивания: `=`, `+=`, `-=`, `++`, `--`
- условные операторы: `if`, `if-else`
- циклы: `while`, `for`
- операторы вывода: `print`, `println`
- работа со списками: `list`, `new`
- блочная структура: `{ ... }`

---

## Архитектура проекта

Проект состоит из следующих компонентов:

- `lexer.l` - лексический анализатор, описывающий токены языка
- `parser.y` - синтаксический анализатор, содержащий грамматику языка
- `lex.yy.c` - сгенерированный лексер
- `parser.tab.c`, `parser.tab.h` - сгенерированный парсер
- модули генерации Python-кода

Лексер преобразует входной текст в поток токенов, после чего парсер выполняет синтаксический анализ и управляет процессом генерации выходного кода.

## Сборка проекта

### Требования

| Инструмент | Назначение | Ссылка |
|:------------:|:------------:|:--------:|
| Flex | Генерация лексического анализатора | https://gnuwin32.sourceforge.net/packages/flex.htm |
| Bison | Генерация синтаксического анализатора | https://gnuwin32.sourceforge.net/packages/bison.htm |
| Python | Запуск сгенерированного Python-кода | https://www.python.org/ |

### Сборка

1. Клонируйте репозиторий:
```
git clone https://github.com/vadimHey/Translator.exe.git
```
2. Убедитесь, что установлены все необходимые инструменты (Flex, Bison, Python).
3. Откройте терминал в папке с проектом и выполните команды для сборки транслятора:
```
bison -d parser.y
flex lexer.l        
gcc -o translator lex.yy.c parser.tab.c
```
4. При запуске транслятора введите название входного файла:
```
./translator.exe
Enter file name: input.txt
```

### Пример

Входной код:
```c
void main() {
    int x = 5;
    x++;
    print(x);
}
```

Выходной Python-код:
```python
x = 5
x += 1
print(x)
```
